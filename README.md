Esse projeto esta sendo criado para treinar os fundamentos de DevOps a medida em que vou aprendendo, a ideia é criar um projeto "amador" e ao longo do tempo agregar qualidade ao mesmo. Estou desenvolvendo o frontend e o backend para ser minimamente funcional e posteriormente testar deploy em cloud.

As tecnologias que estou utilizando são as seguintes:

Front-end: Next.js - É um pouco menos popular e flexível mas, tende a ser bem robusto, com roteamento automático, otimização de imagens, SSR e SSG.

Back-end: NestJS - Um pouco mais complexo, porém é mais organizado, mais escalável e pode ser desenvolvido microserviços caso necessário.

Banco de dados: PostgreSQL - Relacional escalável bastante empregado.
